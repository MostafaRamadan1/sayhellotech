#### Job Description

We're hiring a **Full Stack Developer** to design, develop, and maintain robust web applications using Java and Angular. You will work on both frontend and backend systems, creating seamless user experiences while building scalable server-side solutions. Your role is crucial in delivering high-quality software that meets business requirements and user needs.

Daily responsibilities include developing RESTful APIs using Java and Spring Framework, building responsive user interfaces with Angular, writing clean and maintainable code, and collaborating with cross-functional teams including designers, product managers, and other developers. You'll also participate in code reviews, debugging, testing, and deploying applications to production environments.

This position is well-suited for someone who enjoys solving complex problems and thrives in a collaborative environment. By leveraging modern development practices and staying current with technology trends, you will play a vital role in building innovative solutions that drive business growth.

##### What you'll bring

As a Full Stack Developer, you'll be expected to combine strong backend and frontend skills to deliver end-to-end solutions. You should be comfortable working in an agile environment, writing clean code, and collaborating with cross-functional teams to build exceptional products.

- Strong proficiency in Java, Spring Framework.
- Extensive experience with Angular and TypeScript.
- Solid understanding of RESTful API design and microservices architecture.
- Experience with SQL databases such as PostgreSQL, MySQL, etc.
- Knowledge of ORM frameworks (JPA, Hibernate).
- Knowledge of (Git).
- Ability to write unit and integration tests.
- Strong problem-solving skills and attention to code quality.
